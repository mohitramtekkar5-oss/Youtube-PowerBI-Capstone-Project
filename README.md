# Business Intelligence Capstone Project

## 📌 Project Overview
This repository contains my Capstone project, representing the culmination of my data analytics training. This project focuses on high-level data modeling and sophisticated visualization techniques to provide actionable business insights from raw data.

## 🛠️ Tech Stack
* **Visualization:** Power BI (Advanced)
* **Modeling:** Star Schema / Snowflake Schema design
* **Advanced DAX:** Time Intelligence, Ranking, and Dynamic Filtering

## 🔍 Core Competencies Demonstrated
* **Data Transformation:** Performed advanced cleaning and normalization to ensure data accuracy.
* **Problem Solving:** Translated high-level business questions into specific, measurable dashboard KPIs.
* **UI/UX Design:** Focused on user-centric design principles to ensure the dashboard is intuitive for stakeholders.
